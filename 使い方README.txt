# ビストロパパ 料理写真管理ツール 使い方

## 初回セットアップ（最初の1回だけ）

1. `セットアップ.bat` をダブルクリック
2. 完了メッセージが出たらOK

---

## 毎日の使い方

### STEP 1: 起動
`起動する.bat` をダブルクリック

### STEP 2: フォルダを開く
「📂 フォルダを開く」をクリック → SDカードのフォルダを選ぶ

### STEP 3: フィルタ実行
「🔍 フィルタ実行」をクリック
→ ブレ・暗すぎ・明るすぎの写真を自動除外
→ 候補写真がビューワーに表示される

### STEP 4: 写真を選ぶ（ここだけ手動）
| キー | 動作 |
|------|------|
| **Enter** | ✅ この写真を選択（緑枠） |
| **Delete** | ❌ スキップ（赤枠） |
| **← →** | 前後の写真へ移動 |

### STEP 5: 処理実行
「✅ 選択画像を処理する」をクリック
→ AIが料理名を自動判定
→ ファイル名を自動リネーム
→ 出力フォルダに保存

**完成ファイル名の例：**
```
肉じゃが_横_20260309_001.jpg
パスタカルボナーラ_縦_20260309_002.jpg
唐揚げ_横_20260309_003.jpg
```

### STEP 6: Canvaにアップ
出力フォルダを開いて、Canvaのテンプレートに写真をドラッグ&ドロップ

**テンプレート（ブックマーク推奨）：**
- 横型: https://www.canva.com/design/DAG82rf3QdU/Nh9VI4uDxPbSpQQrWRn3aQ/edit
- 縦型: https://www.canva.com/design/DAHBJ3vaBZs/O9kfWtxtaZzbxYaFTvo3Zg/edit

---

## 設定

「⚙️ 設定」から変更できます：
- **Anthropic API Key**：料理名の自動判定に必要（https://console.anthropic.com）
- **出力フォルダ**：デフォルトはデスクトップの `bistropapa_output`
- **ブレ検出閾値**：数値を小さくすると厳しくなる（推奨:100）

---

## よくある質問

**Q: 料理名の判定が間違っていた**
A: 出力ファイルを手動でリネームしてください。次バージョンで修正機能を追加予定です。

**Q: フィルタで良い写真が除外された**
A: 「フィルタ実行」後、除外された写真は上部のログに表示されます。必要な場合は閾値を調整してください。

**Q: Canvaのテンプレートが500枚になった**
A: Canvaで新しいテンプレートをコピー作成し（BistroPapa横クレジット14など）、URLを控えておいてください。
