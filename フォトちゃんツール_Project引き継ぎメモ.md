## ツール概要

**ツール名：** bistropapa_tool.py  
**目的：** 毎朝のDSLR食材写真→Canva Pro投稿までのワークフローを自動化  
**動作環境：** Windows デスクトップ Python

**主な処理フロー：**
1. SDカードから写真を選択
2. 料理名を手動入力
3. OpenCVで画像品質フィルタリング＋縦横判定
4. `料理名_縦or横_日付_連番.jpg` 形式にリネーム
5. Canva Proテンプレートへアップロード

**Canvaテンプレート：**
- 横位置：`BistroPapa横クレジット13`
- 縦位置：`BistroPapaクレジット縦9`

**使用ライブラリ：**
- OpenCV（`np.fromfile` / `cv2.imdecode` で日本語ファイル名対応）
- Anthropic API
- Python（Windows環境）
